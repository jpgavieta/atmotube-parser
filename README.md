# atmotube-parser
Python-based pipeline to parse Atmotube data (from PRO or PRO 2) using keyword-based detection into subdivided data frames (or save as a merged CSV)  for spatial or time series analysis.

Features:
- Keyword-based column detection (robust to messy exports)
- Timezone and local time inference from GPS coordinates
- Breakdown and some notes of each sensor used by Atmotube PRO/PRO2

## How to use
1. Upload raw CSV file to 1.upload
2. Run the .py script in 2.make
3. Save the parsed CSV to 3.save
